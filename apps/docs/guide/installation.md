# Installation

## Requirements

- **TypeScript/JavaScript:** Node.js 18+ (22+ recommended)
- **Python:** Python 3.11+

## TypeScript / JavaScript

::: code-group
```bash [npm]
npm install jamaica
```
```bash [pnpm]
pnpm add jamaica
```
```bash [yarn]
yarn add jamaica
```
:::

Or install individual packages:

```bash
npm install jamaica-trn jamaica-parishes jamaica-currency
```

## Python

```bash
pip install jamaica
```

Or with uv:

```bash
uv add jamaica
```

## CLI

```bash
npm install -g jamaica-cli
```

Verify installation:

```bash
jamaica --version
```
